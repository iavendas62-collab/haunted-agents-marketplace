```bash

```

## Setup

After installation, configure your Supabase credentials:

1. Create a `.env` file in your project root:
```env
VITE_SUPABASE_URL=your-project-url
VITE_SUPABASE_ANON_KEY=your-anon-key
```

2. The MCP server will automatically connect to your Supabase project
